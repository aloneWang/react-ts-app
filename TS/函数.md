## 定义函数类型
+ 具名函数
  function add(x:number, y:number): void {}
+ 匿名函数
  let fn = function(x:number, y:number):void {}

书写完整函数类型
```
let fn: (x:number, y?:number) => void = function(x:number, y?:number):void {}

```
`?` 是表示参数是可选的， 可选参数必须跟在 必须参数后面，保持位置一致

## 重载

处理流程： 编译器会查找重载列表， 优先是用第一个，不匹配的话继续下一个
```
function pickCard(x: {suit: string; card: number; }[]): number;
function pickCard(x: number): {suit: string; card: number; };
```
